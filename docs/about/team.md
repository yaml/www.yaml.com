# Team

## Leadership

### Ingy döt Net

**Founder**

![Ingy döt Net](https://github.com/ingydotnet.png){ width="200" }

Ingy döt Net is the co-creator of YAML and founder of The YAML Company. Along with Clark Evans and Oren Ben-Kiki, Ingy helped design and specify YAML starting in 2001.

#### Background

- **Co-created YAML** (2001) - Worked with Clark Evans and Oren Ben-Kiki to design the YAML data language
- **Open Source Pioneer** - Decades of contributions to Perl, Python, and other communities
- **ActiveState Alumni** - Worked on Perl core and data serialization in the early 2000s
- **Created YAMLScript** - Designed the next-generation YAML programming language

#### YAML Projects

Ingy has created and maintains numerous YAML-related projects:

- [go-yaml](https://github.com/yaml/go-yaml) - YAML for Go
- [pyyaml](https://github.com/yaml/pyyaml) - YAML for Python
- [libyaml](https://github.com/yaml/libyaml) - Fast C YAML library
- [YAMLScript](https://yamlscript.org) - YAML programming language
- [yaml-test-suite](https://github.com/yaml/yaml-test-suite) - Comprehensive YAML test suite
