## Learning YAML

### For Beginners

- Start with the [FAQ](faq.md) to understand the basics
- Use the [Cheatsheet](cheatsheet.md) as a quick reference
- Read the [Glossary](glossary.md) to learn YAML terminology

### For Implementers

- Read the [YAML 1.2.2 Specification](https://yaml.org/spec/1.2.2/)
- Join the [YAML Matrix Chat](https://matrix.to/#/#chat:yaml.io) for discussions

  - [go-yaml](../projects/go-yaml.md) (Go)
